### General Info
***
El objetivo de este proyecto es  Aplicar técnicas de validación cruzada y entender  cuando usar diversas estrategias diferentes de validación cruzada:KFold, StratifiedKFold, GroupKFold.

<b>Objetivos:</b> 
- Entender los peligros del hold out
- Aplicar técnicas de validación cruzada (cross validation)
- Utiliza la aleatoriedad a tu favor
- Entender cuando usar diversas estrategias diferentes de validación cruzada
KFold, StratifiedKFold, GroupKFold
- Trabajar en la previsión de nuevos grupos cuando no todos los datos son lecturas independientes entre si
- Utilizar un pipeline para entrenamiento y validación
